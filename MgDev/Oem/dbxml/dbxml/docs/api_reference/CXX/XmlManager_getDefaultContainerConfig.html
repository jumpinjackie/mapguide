XmlManager::getDefaultContainerConfig 
#include <DbXml.hpp>


XmlContainerConfig XmlManager::getDefaultContainerConfig()
Description: XmlManager::getDefaultContainerConfig

Returns a copy of the default container configuration used by calls to XmlManager::openContainer and XmlManager::createContainer by this XmlManager object. Use XmlManager::setDefaultContainerConfig to set the default XmlContainerConfig.
