XmlContainer::getFlags 
#include <DbXml.hpp>


XmlContainerConfig XmlContainer::getFlags() const;
Description: XmlContainer::getFlags

The XmlContainer::getFlags method returns a copy of the XmlContainerConfig with the settings used by XmlManager::createContainer or XmlManager::openContainer.

This method is deprecated in favor of XmlContainer::getContainerConfig.
