XmlContainerConfig::setExclusiveCreate 
#include <DbXml.hpp>


void XmlContainerConfig::setExclusiveCreate(bool value)


bool XmlContainerConfig::getExclusiveCreate() const
Description: XmlContainerConfig::setExclusiveCreate

If set to true XmlManager::openContainer and XmlManager::createContainer will throw exceptions if the container already exists and XmlContainerConfig::setAllowCreate has been set to true. The default value is false.

Parameters
value
  • Set to true to get exclusive create semantics.
