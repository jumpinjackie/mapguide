About External Files

You can connect to external SDF, SHP, and raster files, without loading them into the MapGuide Library.

You have the option of entering a pathname or selecting a file using the aliases set up in Site Administrator. If you enter a pathname, the file must be located either on the Infrastructure Map Server, although not in the resource repository, or in a location directly accessible from the server. If you want to access resources on your local machine, you must set up an alias in Site Administrator; entering the pathname to a location on your local machine will not work.
