The MapGuide Viewer Interface
 
 

The MapGuide Viewer interface may include the following sections:

  • The Layers Pane displays the layers and layer groups in the map. Right-click the list of layers to access commands related to the layer list and the display of map layers.
  • The Properties Pane displays properties of selected items. In the DWF Viewer, right-click the Properties Pane for additional options.
  • The Task Pane displays task options and information or help topics, depending on your actions.
