Measuring
 
 
 

Use the Measure command to measure distances on the map.

To measure distances

  1. Choose the Measure command.

    The Measure Distance task will appear in the Task Pane on the right side of the viewer window.

  2. Click the map to specify the start and end points.
NoteTo change the units of measure, see Changing Viewer Options.
