Panels

The Web Studio interface separates functionality into 'panels'. Panels are stacked vertically on the left and the right hand side of the application. On the left side, there are two panels: Site Explorer and Open Resources. On the right hand site, the panels reflect the current open resource.

All panels work in the same way. Each panel has:

  • A busy indicator, which is visible when the panel is waiting for asynchronous content to load.
  • A title.
  • An optional help icon which opens the embedded help system with a help page for the appropriate panel.
  • A minimize icon which will collapse the panel. When a panel is collapsed, its space is given to the next available panel above the one that is collapsed. If no panel is open above the one that is collapsed, then a panel will be opened. The exception to this rule is the top-most panel, which will give its space to a panel below it.
  • A maximize icon, which will expand the panel to take up all available space and will collapse all other panels.
  • An optional close icon, which is used on the right hand side in the top most panel to close an open resource.
