Map Properties

The map properties panel shows non-editable information about a map definition:

Resource Name
This is the name of the map resource as it appears in the tree view
Type
This is the type of the resource, which is always MapDefinition for maps.
Location
This is the path of the resource in the repository
Owner
This is the name of the user that currently owns the resource. By default, this will be the name of the user that created the resource
Created
The date and time that the resource was created.
Modified
The last date/time that the resource was modified.
Resource References
There are two tabs which list the references related to this resource. The References tab lists resources that this resource refers to. If those resources are moved or deleted, this resource will become invalid. The Referenced By tab lists resources that refer to this resource. If this resource is moved or deleted, those resources will become invalid.
