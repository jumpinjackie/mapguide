Open Site Dialog

The open site dialog is used to connect to a MapGuide Open Source Site. Enter a URL, user name and password and click OK to connect to a server.

URL
The URL to a MapGuide Open Source server. The URL is in the form http://server/mapguide and is the URL to the base installation of your Web Tier. If you are unsure what to put here, contact your administrator.
Username
Enter the user name provided by your administrator. The Username is case-sensitive
Password
Enter the password provided by your administrator. The password is case-sensitive

The Open Site dialog appears automatically when you first load Web Studio. Once you have connected to a site, you will have to disconnect before you can connect to another site.
