Style Area Dialog

The style area dialog is opened when you click on a style preview in the Layer Styles panel and the geometric type of the feature class is polygonal. The style area dialog allows you to configure the fill and border styles of an area style.

Fill

To enable a fill style, select a fill pattern. Selecting a fill pattern of 'No Fill' will disable filling of polygons. You can set the following properties of a Fill:

Pattern
Select a fill pattern by clicking the [...] button to the right of the fill pattern. Selecting 'No Fill' disables fills for polygons.
Foreground Color
Enter an 8 digit hex code (AARRGGBB) for the foreground color. The foreground color will be used wherever the pattern is black.
Background Color
Enter an 8 digit hex code (AARRGGBB) for the background color. The background color will be used wherever the pattern is white.

Border

To enable a border style, select a border pattern. Selecting a border pattern of 'No Border' will disabled borders on polygons. You can set the following properties of a Border:

Pattern
Select a border pattern by clicking the [...] button to the right of the border pattern. Selecting 'No Border' disables borders for polygons.
Thickness
Enter a number for the width of the border. Use 0 to have the thinest line available (typically 1 pixel wide). The thickness of a line is also determined by the units selected.
Color
Enter an 8 digit hex code (AARRGGBB) for the border color. The border color will be used wherever the pattern is white.

Units

Units
This controls the units used when interpreting the thickness of a border. Select from Points (72 points per inch), Millimeters, Inches or Centimeters.
Size Context
This indicates what the units are relative to. The options are map space and device space. By default device space is used.
