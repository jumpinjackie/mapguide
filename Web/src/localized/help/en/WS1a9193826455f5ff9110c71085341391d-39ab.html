Selecting Within
 
 

After you have selected an area, such as a buffer or a polygon on the map, you can select certain features within that area.

To select features within a buffer or another feature

  1. Select map features using another selection command, such as Buffer.
  2. Choose the Select Within command.
  3. The Select Within task will appear in the Task Pane on the right side of the viewer window.
  4. Under Restrict Results To Selected Layers, click one or more layers containing the features you want to select.

    All features within the previously selected area will have their layers displayed in the layer list. By default, the first layer is selected. If you select one or more additional layers, adding more features, click Refresh to manually update the list.

  5. Click Done.

    This will select all features within the highlighted area that are located on the specified layers.

NoteClick Clear to remove all selections from the map and all layers from the list.
