Available Versions of MapGuide Viewer
 
 
 

MapGuide Viewer is available in three versions:

  • AJAX Viewer - a browser-neutral viewer that does not require a plug-in
  • Fusion-based AJAX Viewer - uses the popular Open Layers project for both non-tiled and tiled maps
  • DWF™ Viewer - a Microsoft® ActiveX® control that works only in Microsoft Internet Explorer

The map author specifies the version of the viewer when creating the map. The author can create a version of the map for either one or both viewer versions. If the map is available for both viewers, you might notice a few minor differences in functionality between the viewers. For a list of these differences, see Differences Between Viewer Versions.

NoteYou can uninstall the DWF Viewer. See Uninstalling the DWF Viewer.
