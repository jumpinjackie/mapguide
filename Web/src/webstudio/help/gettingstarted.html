Welcome to MapGuide Open Source Web Studio

1. Find Data

MapGuide references data through Feature Sources. There are two ways of loading file-based data into a Feature Source: Uploading managed data and referencing unmanaged data.

Upload Data into a new Feature Source
Create new Feature Source and upload .SHP or .SDF data files into it. The data will then be managed by MapGuide.
Reference Unmanaged Data in a new Feature Source
Create a new Feature Source and reference .SHP or .SDF data files. If files are already on the server, but not managed by MapGuide they can be referenced by providing the path to the data. The data will remain unmanaged by MapGuide, but will be available to make layers and maps.

2. Build Layers

Create a new Layer
Layers add style to the data. Feature Sources need to be linked to Layers so the data can be styled. Layers can be themed and labeled by data attributes to give them meaning. They can be themed and labeled by scale also to control their appearance at any scale.

3. Make a Map

Create a new Map
Maps are a collection of Layers. Once styled, Layers can be used together to build up a map. Layers should be put in order, with points on top, lines next, and polygons and rasters on the bottom.
