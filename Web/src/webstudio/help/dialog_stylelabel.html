Style Label Dialog

The style label dialog is opened when you click on a label preview in the Layer Styles panel. The style label dialog allows you to configure the style of a label. To enable labels, select a property to display using the select list or the expression builder. To remove labels, select 'None' in the 'Property to display' list.

Label Properties

Property to Display
Select a property using the select list, or click the [...] button to the right of the select list to open the Expression Builder dialog. If you choose to use an expression, build a text expression
Font
Type the name of a font to use for the label. The font must be installed on the server. If you do not know which fonts are installed on the server, ask your administrator. Font names may be case-sensitive. If you type the font name incorrectly, the server will select a default font.
Size
The size, in points, of the label. The size can be fixed (enter a number) or derived from an attribute of the data. To use an attribute, click the [...] button to the right of the Size and enter a number expression
Format
Control the format of the font by checking or unchecking the bold, italic and underline check boxes.

Style Properties

Text Color
Enter an 8 digit hex code (AARRGGBB) for the color to render the label in.
Background Style
The background style determines how the background of the label is drawn. Transparent means there is no background. Opaque means the background is drawn as a solid rectangle in the background color. Ghosted means an outline will be drawn around the characters of the label in the background color.
Background Color
Enter an 8 digit hex code (AARRGGBB) for the background color. How the background color is used is determined by the background style.

Orientation Properties

Fixed
This property is only enabled for point layers. If this option is selected, the label can be positioned in a fixed way relative to its symbol using the Horizontal and Vertical alignment. If this option is not selected, Horizontal and Vertical alignment are not available as the label is positioned automatically by the server.
Horizontal Alignment
For point layers that have Fixed label placement, select how the label will be positioned relative to its symbol. You can use a position from the select box, or use the [...] button to open the expression builder dialog and select an attribute. Note that this field only accepts an attribute name, not a true expression.
Vertical Alignment
For point layers that have Fixed label placement, select how the label will be positioned relative to its symbol. You can use a position from the select box, or use the [...] button to open the expression builder dialog and select an attribute. Note that this field only accepts an attribute name, not a true expression.
Rotation
This controls the rotation of the label with 0 being a horizontal label. You may enter a value between 0 and 360, or use the [...] button to open the expression builder dialog and enter a number expression.

Units

Units
This controls the units used when interpreting the size of a font. Select from Points (72 points per inch), Millimeters, Inches or Centimeters.
Size Context
This indicates what the units are relative to. The options are map space and device space. By default device space is used.
