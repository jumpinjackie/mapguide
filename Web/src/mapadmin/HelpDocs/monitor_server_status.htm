Monitoring the Status of a Server

 
You can view information about the status of any server in the site.

Viewing processor usage and up time

To monitor server status

1 On the Manage Servers page, select the server you want to monitor.

2 Click Status.

Details about the selected server appear in a separate window. The log window has controls to:

  • Refresh the status.
  • Autorefresh at a specified interval.
