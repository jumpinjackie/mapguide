Performance Report

Creating a Performance Report

Use the Performance Report feature in Site Administrator to view performance statistics on a map definition and its underlying layer definitions. This allows you to quickly identify and troubleshoot performance issues.

To create a Performance Report:

  • 1. In Site Administrator, select Performance Report.
  • 2. In the Settings section, do one of the following:
    • To define a new report, select the desired map resource name. The path to the map definition is displayed beside the selected resource name.
    • OR
    • To use settings for a recent report, select the report you want under Recent Settings. Skip the next step.
  • 3. Click Select Settings to select a center point and scale in the map dialog. Or, you can enter the values manually in the Center Point and Scale fields.
  • 4. When the Center Point and Scale fields contain the desired values, click Run to generate the Performance Report.
