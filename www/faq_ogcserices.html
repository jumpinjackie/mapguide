1. What OGC services does MapGuide Open Source support?
MapGuide Open Source supports the WMS and WFS services. The map agent directly supports the OGC request formats for these two services. In addition MapGuide Open Source will support consuming WMS and WFS data sources as soon as the FDO providers for WMS and WFS are available.
2. What versions of WMS are supported?
Currently MapGuide Open Source supports WMS 1.0.0 and 1.1.1.
3. What versions of WFS are supported?
Currently MapGuide Open Source supports WFS 1.0.0.
4. Is the WMS and WFS functionality certified by OGC?
Not yet, but we are going through the exercise of running OGC test suite in preperation for WMS 1.1.1 and WFS 1.0.0 certification.
