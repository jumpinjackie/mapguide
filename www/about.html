Project

Mission Statement

To create, as a community, the leading international web-based platform for developing and deploying web mapping applications and geo-spatial web services. A platform that is derived from service-orient architecture concepts, built on open source components, supports rich access to geo-spatial data, provides a full suite of geo-spatial processing capabilities, offers visually stunning cartographic capabilities, and flexible options to support any browser on any platform.
