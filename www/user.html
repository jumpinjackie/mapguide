MapGuide Open Source - Getting Involved as a User

There are two different levels of involvement for users and application developers depending on whether you are a registered user or not. Unregistered users can download and install the software, join and participate in the mailing lists (using the links below), browse the mailing list archives, browse the source code in Subversion, and access all of the documentation and information on this site.

If you register as a user of the osgeo.org website and become a member of the MapGuide Open Source project, you can directly manage your mailing list subscriptions using the tools on this site and use the Project Tracker tool to browse and enter defects and enhancement requests. You can register as user on the osgeo.org site by selecting the Register link at the top right corner of the Open Source Geospatial Foundation banner. Once you are a registered user, you can request to become a project member with a user role using the Request project membership/role link at the top of the MapGuide Open Source project home page.

Join the Mailing Lists

Join the mailing lists to learn what other users are doing with MapGuide Open Source, ask questions, get answers, and when you get comfortable jump in and answer questions. The users@mapguide.osgeo.org mailing list provides a wealth of information for all MapGuide Open Source users from novices to experts. Find out about new releases and events related to MapGuide Open Source on the announce@mapguide.osgeo.org mailing list. If you are a registered user click here to sign up. If you are not a registered user, use one of these links:

  • Subscribe to the users list: users-subscribe@mapguide.osgeo.org
    or the users digest: users-digest-subscribe@mapguide.osgeo.org
  • Subscribe to the announce list: announce-subscribe@mapguide.osgeo.org
    or the announce digest: announce-digest-subscribe@mapguide.osgeo.org
To browse archives of any of the mailing lists, select the Mailing lists link under Project tools.

Entering Defects and Enhancements

As a register user and member of the MapGuide Open Source project, you can use the Project Tracker tool to browse and enter defects and enhancement requests. You can access the Project Tracker tool from anywhere on the site using the link under Project tools.
