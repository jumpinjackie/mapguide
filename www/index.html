Description

MapGuide Open Source is a web-based platform that enables users to quickly develop and deploy web mapping applications and geo-spatial web services. Web mapping applications built on MapGuide feature a highly interactive viewer that includes support for feature selection, property inspection, map tips, and operations such as buffer, select within, and distance calculation. MapGuide makes extensive use of XML for defining content and supports the poplular geo-spatial file formats, databases, and standards. The MapGuide platform runs on both Linux and Microsoft Windows, supports Apache and IIS web servers, and supports application development in PHP, ASP.NET, Java, and JavaScript.

MapGuide Open Source is free, open source software licensed under the LGPL.
