MapGuide Open Source - System Requirements

MapGuide Viewer

MapGuide Open Source offers two viewer technologies: A DWF™-based viewer and an AJAX-based viewer.

  • The Autodesk DWF Viewer supports Windows 98 SE, Windows 2000 (SP2), Windows XP Professional and Home Edition, and Internet Explorer 5.01 and later.
  • The AJAX Viewer uses HTML and DHTML to deliver maps and spatial data to the browser. It does not require a plug-in and supports most browsers, including Microsoft Internet Explorer, Firefox™, and Safari browsers.

MapGuide Server

  • Linux: Red Hat Enterprise ES 3.0
  • Windows: Windows Server 2003 and Windows 2000

Web Servers

  • Apache HTTP Server 2.0
  • Microsoft IIS 5.0 on Windows 2000
  • Microsoft IIS 6.0 on Windows Server 2003
  • Tomcat Servlet engine version 5.5.9

Scripting Languages

  • PHP
  • .NET
  • Java™ (JSP)

There is 100 percent consistency among the API (application programming interface) calls for each language.
