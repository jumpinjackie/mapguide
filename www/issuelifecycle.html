Issue Status

The Status of an issue indicates where in its lifecycle the issue is. Only certain state transitions are allowed depending on the user's role. The Resolution field indicates how a particular issue has been addressed.

Unconfirmed   Initial state of an issue that has been entered by a User. A Project Developer, Manager, or Owner is responsible for validating the issue and changing it's state to New, Assigned, or Resolved.
New   Initial state of an issue that has been entered by a Project Developer, Manager, or Owner.
Assigned   The issue has been accepted and assigned to an individual.
ResolutionProposed   A resolution has been proposed by a Project Contributor. A Project Developer, Manager, or Owner is responsible for reviewing the proposed resolution and either accepting it or sending it back for additional work.
Resolved   A resolution to the issue has been reached and is awaiting verification from quality assurance.
Verified   Quality assurance has looked at the issue and the resolution and agrees that an adequete resolution has been reached. Issues remain in this state until a version of the project with the resolution is released.
Closed   The issue is considered resolved and a version of the project has been released.
Reopened   The issue was once resolved, but the resolution was deemed incorrect.

Issue Resolutions

Issues in the Resolved state must have one of the following resolution values.

Submitted   A resolution to this issue has been submitted to the source code repository and tested.
Invalid   The problem described is not an issue or no longer an issue.
WillNotAddress   The problem described is an issue which will never be addressed.
Defer   The problem described is an issue which will not be fixed in this release cycle.
External   The problem described cannot be addressed due to problems or limitations in third party components for which we have no control.
Duplicate   The problem described is a duplicate of another issue.
NotReproducible   The problem as described cannot be reproduced using the information given.
Moved   The issue has been moved to another project where it can be addressed.
