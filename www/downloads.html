Source Code

Use these links to configure, make, and install MapGuide Open Source on Linux. Read the Building MapGuide on Linux document for instructions on how to get it up and running.
  • mapguideopensource-1.0.0.tar.gz [68,710 kb]
    md5: 85ded9229c79b452ecca6044a6f4c450
    Compressed tar file that contains source code for both the MapGuide Server and MapGuide Web Server Extensions with configure / make / make install scripts. Requires the FDO API which is available from fdo.osgeo.org.
  • mapguideaptinstaller-1.0.0.tar.gz [18,443 kb]
    md5: 1408690a49ac5a3438c95a3631ece672
    Compressed tar file that contains source code for the Apache Web Server, PHP, and Tomcat with shell scripts to automatically build and configure these tools for use with MapGuide Open Source.

Windows Installers

Use these links to install MapGuide Open Source on Microsoft Windows. Read the Installing MapGuide on Windows document for instructions on how to get it up and running.
  • MgServerSetup.exe [14,947 kb]
    md5: 081d23296d0cdda3d858ca3e591d5de4

  • MgWebServerExtensionsSetup.exe [31,464 kb]
    md5: 4cdc2aa1643bccdbd8167135df5cceff
    Self extracting executable that will install the MapGuide Web Server Extensions on Microsoft Windows with the option to either install the Apache Web Server, PHP, and Tomcat or to configure an existing Microsoft IIS installation.
  • Additional FDO Provider Previews
    Additional FDO Providers for ArcSDE, MySQL, ODBC, WMS, WFS, and Raster Files (Windows Platform Only). See the readme.txt file included with each provider package for installation instructions.
