MapGuide Open Source - Features

MapGuide Open Source is a web-based platform that enables users to quickly develop and deploy web mapping applications and geospatial web services.

  • Interactive Map Viewing
    • AJAX Viewer or DWF Viewer with a common user interface and JavaScript based API
    • Tiled map display featuring smooth pan and zoom (AJAX Viewer only)
    • Feature selection by point, rectangle, radius, or polygon with immediate feedback
    • Multi-line tooltip display of feature properties
    • Property browser provides immediate display of the property values of a selected feature
    • Support for feature buffering, query of features that fall within selected polygon features, and the ability to measuring distance
    • Printing and plotting support

  • Quality Cartographic Output
    • Scale dependent detail
    • Rule driven thematic layer stylization
    • Dyanmic labeling with overpost detection, labels that follow the path of a line, and automatic label resizing
    • Anti-aliased image rendering
    • True color support
    • TrueType font support

  • Built-in Resource Database
    • Provides hierarchical storage of XML resource documents defining maps, layers, data connections, symbology, and viewer layout
    • Allows linkage and reuse of resources, e.g. define data sources and layers once then reference them from multiple maps
    • Access based security model with inheritable permissions
    • Permits meta-data and binary data to be associated with resource documents

  • Uniform Data Access
    • Exclusive use of the Feature Data Objects (FDO) API for vector and raster feature data access
      • Support for ESRI SHP and SDF (available now)
      • Support for ESRI ArcSDE, MySQL, ODBC, GDAL Raster, WMS, and WFS (coming soon)
    • Exclusive use of DWF for CAD based data access

  • Flexible Application Development
    • Write your application logic within the web server environment to work with either viewer on any client
    • Support for application development in PHP, .NET, or Java

  • Extensive Server-Side APIs
    • Create, query, read, and write permanant and session based XML resource documents
    • Feature data query and update, including spatial and attribute query support, and schema discovery across all supported data sources
    • Coordinate system factory, definitions, coordinate reprojection, and great circle distance computation
    • Geometry creation, manipulation, and analysis with support for spatial predicates based on the DE-9IM model, overlay functions (intersection, union, difference, symetric difference), buffer, convex hull, area, and distance functions
    • Vector and raster map stylization and rendering
    • Runtime map and layer manipulation, add / remove / change layers, change layer and group visibility, modify the definition of a layer, and alter the current center and scale

  • Fast, Scalable, and Secure Server Platform
    • Fully multi-threaded and takes advantage of multi-core / multi-processor technology
    • Multiple servers can be confugured as one logical site to improve scalability
    • FastCGI based Map Agent loads once and stays loaded for use with both Apache and Microsoft IIS
    • Database connections are pooled and shared, greatly improving performance
    • Tile caching ensures map image tiles are only rendered once for the AJAX Viewer no matter how many users are viewing map
    • Connection oriented, role based security
    • Configurable logging of client access, administrative access, authentication attempts, errors, session lifetimes, and request tracing
    • Web-based site administration allows remote site and server configuration and log file access

  • Cross Platform
    • Server can be deployed on Microsoft Windows as a Service or on Linux as a Daemon
    • Supports Apache on Windows or Linux, and IIS on Microsoft Windows
    • Web browser support for Microsoft Internet Explorer, Mozilla Firefox, and Sarfari

  • Support for Open Geospatial Consortium Standards
    • Web Map Service
    • Web Feature Service
