MapGuide Open Source 1.1 Readme

Thank you for using MapGuide Open Source 1.1.

This release is fully compatible with resources created using the MapGuide Open Source 1.0 releases. If you uninstall the 1.0 release and then install this one in the same place, your existing Resource Database will be preserved.

Any data/resources created with the MapGuide Open Source 1.1 release will not necessarily be compatible with future versions of MapGuide Open Source.

There are some API changes that will require changes to .NET and Java applications. Please see the limitations section below for details.

Minimum Hardware Requirements

The MapGuide Open Source 1.1 release requires the following:

  • Server: 1 GHz Intel Pentium III or higher processor recommended, 1GB of RAM
  • Web Tier: 1 GHz Intel Pentium III or higher processor recommended, 1GB of RAM
  • DWF Viewer: 800 MHz Intel Pentium III or higher processor recommended, 128MB of RAM
